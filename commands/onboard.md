Initialize a claudity-assurance QA environment for a new project. This command
invokes the `c-a:onboard` skill which conducts a collaborative interview to
understand the system under test, scaffolds the QA directory structure, writes
initial documentation, and configures the interaction surface.

## Invoke the Onboard Skill

Use the Skill tool to invoke `claudity-assurance:onboard`. Follow its full
workflow — interview, scaffold, initial docs, interaction layer configuration,
and optional BDD seeding.

The skill contains the complete onboarding process. Refer to it for all details.
